## 本地部署

拉取 WIKI 项目

```
git clone https://github.com/Mueoink/zsyx-wiki.git
```

安装依赖

```
pnpm install
```

启动 WIKI
```
pnpm docs:dev
```

构建
```
pnpm docs:build
```
